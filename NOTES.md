## DOME ID

As of version x.x, the DOME ID contains the following information:
1. Region (REG_ID)
2. Stratum (STRATUM)
3. Climate ID (CLIM_ID)
4. RAP (RAP_ID)
5. Management (MAN_ID)
6. RAP Version (RAP_VER)
7. Description (DESCRIPTION)
